Welcome to clockmod, a modified version of clockwork

This version focuses more on UI an more builtin apps, so you feel like this is a REAL computer.

Clockmod uses the same app formatting as clockwork, so its entirelly backwards-compatible. All apps that work on clockwork, work on here.
As well as having complete backwards-compatibility, clockmod also features the exact same app store as clockwork, so you can migrate effortlessly.

Clockmod shares most of the same code as clockwork and will be taken down if the creators of clockwork (Redstone Network) request me to do so. Please send it to me via email.
